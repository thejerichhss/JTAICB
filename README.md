## JTAICB
*JTAICB*, (or *Jerich's Terminal AI Chatbot*) is **Gemini, but in a terminal.** (Works with WSL!) 
###How to Run:
####1. Clone or Download JTAICB. 
```bash
git clone https://github.com/thejerichhss/JTAICB.git
```
####2. Run DEPENDENCIES.sh, 
```bash
chmod +rwx DEPENDENCIES.sh, ./DEPENDENCIES.sh
```
####3. Replace "*GEMINIAPIKEY_PLACEDHERE*" in the JTAICB.sh file with a *"Gemini API Key",* You can get one [**here**](https://aistudio.google.com/apikey)
```bash
sudo nano JTAICB.sh
```
####4. Run JTAICB.sh, 
```bash
chmod +rwx JTAICB.sh, ./JTAICB.sh
```
